# Poke Search

A pokedex made with [Vue](https://vuejs.org/) using [PokeAPI](https://pokeapi.co/)

![image](https://github.com/KonyD/poke-search/blob/main/src/assets/screenshot.png)

# Running The Project

npm:

```bash
npm run dev
```

pnpm:

```bash
pnpm run dev
```

# Building The Project

npm:

```bash
npm run build
```

pnpm:

```bash
pnpm run build
```
